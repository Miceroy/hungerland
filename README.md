# Hungerland - Näläkämoa

Hungerland Game Engine is a simple 2d game engine made with modern C++.

# Cloning repository
```
git clone https://github.com/Miceroy/hungerland.git
```

# Compiling

```
mkdir build
cd build
cmake ../
cmake --build .
```
